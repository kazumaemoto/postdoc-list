### [JSPS海外特別研究員RRA](https://www.jsps.go.jp/j-ab/rra_gaiyo.html)

- 任期：2年
- 給料：年額450〜620万円

- 期間：最大1年

- 期間：2年
- 支給額：月額200,000円
